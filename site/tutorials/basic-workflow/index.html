    基础工作流教程¶

    本教程介绍 PointWorks 的核心工作流：从打开点云文件，经过滤波和分割处理，最终导出成果数据的完整流程。

    概述¶

    典型的点云处理流程如下：

    加载点云 → 预处理（滤波） → 分析（分割/配准） → 导出成果

    本教程使用一份地面激光扫描点云数据，演示完整操作流程。

    第一步：打开点云文件¶

    1.1 加载文件¶

    通过 File > Open 菜单或工具栏的打开按钮加载点云文件。

    PointWorks 支持以下格式：

    格式 扩展名 说明
    LAS/LAZ .las .laz ASPRS 标准激光雷达格式
    PLY .ply Stanford 多边形格式
    PCD .pcd PCL 点云格式
    E57 .e57 ASTM 工业扫描格式
    TXT .txt .xyz .asc 文本点云格式

    大文件加载

    对于超过 1000 万点的点云，PointWorks 自动启用八叉树模式，采用流式加载（每批 50 万点），加载过程中可在进度条中查看进度并支持取消。

    1.2 全局坐标偏移¶

    如果点云使用 UTM 等大坐标系（如 x=500000, y=4000000），加载时会弹出全局偏移对话框：

    • PointWorks 会自动计算建议偏移值（质心坐标）
    • 点击 OK 确认偏移，内部渲染使用偏移后的坐标
    • 保存文件时会自动加回偏移值，确保数据一致性

    坐标精度

    不使用全局偏移可能导致 GPU 渲染时出现精度丢失（点云抖动、卡顿），务必对大坐标数据启用此功能。

    1.3 查看点云信息¶

    加载完成后，在左侧 点云树 中可以看到新添加的点云节点。3D 视图中会自动渲染点云数据。

    • 鼠标左键拖拽旋转视角
    • 鼠标中键拖拽平移
    • 鼠标滚轮缩放

    第二步：滤波处理¶

    原始点云通常包含噪声和离群点，需要先进行滤波处理。

    2.1 打开滤波器¶

    通过 Tools > Filters 菜单打开滤波器工具浮窗。

    2.2 选择滤波类型¶

    PointWorks 提供以下滤波器：

    滤波器 适用场景
    StatisticalOutlierRemoval 去除统计离群噪声点（推荐首选）
    RadiusOutlierRemoval 去除半径内邻居过少的孤立点
    VoxelGrid 体素降采样，减少点云密度
    PassThrough 按坐标范围裁剪
    GridMinimum 2D 网格最小值投影
    LocalMaximum 去除局部最高点

    2.3 统计离群点去除示例¶

    1. 在滤波器类型下拉框中选择 StatisticalOutlierRemoval
    2. 设置参数：
    3. MeanK: 50（用于计算均值的邻居数量）
    4. StddevMulThresh: 1.0（标准差倍数阈值）
    5. 点击 Preview 预览效果（高亮将被移除的点）
    6. 调整参数直到满意
    7. 点击 Apply 应用滤波

    参数调整

    • MeanK 值越大，滤波越平滑但计算越慢
    • StddevMulThresh 值越小，移除的点越多
    • 建议先用保守参数预览，再逐步调整

    2.4 体素降采样¶

    如果点云密度过高，可使用 VoxelGrid 降采样：

    1. 选择 VoxelGrid 滤波器
    2. 设置 LeafSize（体素边长，单位与点云坐标一致）
    3. 预览并应用

    第三步：点云分割¶

    滤波后，可以使用分割工具提取感兴趣的区域。

    3.1 裁剪工具¶

    通过 Tools > Cutting 打开裁剪工具：

    • Box Cutting - 使用包围盒裁剪
    • Polygon Cutting - 在视图中绘制多边形区域裁剪

    3.2 点拾取工具¶

    通过 Tools > Pick Points 打开点拾取工具：

    • 单点拾取查看坐标信息
    • 多边形区域选择批量拾取

    3.3 使用插件分割¶

    通过 Plugins 菜单使用内置分割插件：

    • CSF Ground Filter - 布料模拟地面分割
    • Vegetation Filter - 基于颜色的植被提取

    详见 地面滤波实战 教程。

    第四步：导出成果¶

    处理完成后，导出点云数据。

    4.1 保存点云¶

    在点云树中右键点击要导出的点云节点，选择 Save As：

    • 选择输出格式（LAS、PLY、PCD、E57、TXT）
    • 设置输出路径
    • 点击保存

    4.2 TXT 导出配置¶

    导出为文本格式时，可以配置：

    • 分隔符（空格、逗号、制表符）
    • 输出字段（X, Y, Z, R, G, B, Nx, Ny, Nz 等）
    • 精度（小数位数）

    格式选择建议

    • LAS/LAZ: 推荐，工业标准，支持完整元数据
    • PLY: 适合交换，广泛支持
    • PCD: 适合 PCL 生态
    • TXT: 适合自定义处理和检查

    完整流程回顾¶

    File > Open         → 加载点云（自动处理坐标偏移）
Tools > Filters     → StatisticalOutlierRemoval 去噪
Tools > Filters     → VoxelGrid 降采样（可选）
Tools > Cutting     → 裁剪感兴趣区域
Plugins > CSF       → 地面点分割（可选）
右键 > Save As      → 导出成果
