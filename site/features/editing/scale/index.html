    尺度缩放¶

    沿 X、Y、Z 轴独立缩放点云的坐标值。可用于单位转换（如米到毫米）或非等比缩放。

    入口路径¶

    菜单: Edit > Scale

    源码: src/edit/scale.h

    参数说明¶

    参数 控件 说明
    Scale X DoubleSpinBox X 轴缩放因子。1.0=不变，>1 放大，<1 缩小
    Scale Y DoubleSpinBox Y 轴缩放因子
    Scale Z DoubleSpinBox Z 轴缩放因子

    操作步骤¶

    1. 选中点云后打开 Scale 工具
    2. 设置三个轴的缩放因子：
    3. 等比缩放：三个轴设置相同的值（如 1000 将米转为毫米）
    4. 非等比缩放：各轴设置不同的值
    5. 拖动滑块时自动 Preview 实时预览效果
    6. 确认后：
    7. Apply: 直接修改当前点云坐标（前缀 -scale）
    8. Add: 创建缩放后的新点云副本（前缀 scaled-），保留原始数据
    9. 点击 Reset 恢复到缩放前的状态

    缩放操作以坐标原点 (0,0,0) 为中心进行。若点云距离原点较远，缩放后可能会产生明显的位移。建议先通过 Transformation 将点云移到原点附近再缩放。

    非等比缩放会改变点云的几何形状，通常仅在特定需求下使用（如补偿传感器畸变）。
