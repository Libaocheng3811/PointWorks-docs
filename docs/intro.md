# 欢迎使用 PointWorks

PointWorks 是一款**开源**的专业三维点云处理桌面软件，基于 Qt5、VTK、PCL 构建。

## 核心功能

- **可视化**：VTK 渲染引擎，八叉树 + LOD 自适应渲染
- **滤波处理**：多种滤波算法，支持体素降采样、统计离群点移除
- **配准**：支持中心对齐、全局配准、ICP 精配准
- **分割**：形状检测、区域生长、欧式聚类等
- **变化检测**：云对云/云对网格距离计算
- **Python 脚本**：内嵌 Python 3.9，支持自动化批处理
