# 简介

PointWorks 是一款基于 Qt5、VTK 和 PCL 构建的专业三维点云处理桌面应用程序。

## 功能概览

PointWorks 提供以下核心功能模块：

- **数据管理**：支持 LAS/LAZ/PLY/PCD/E57/TXT/OBJ/STL/VTK 等多种格式的导入导出
- **可视化**：基于 VTK 的三维渲染，八叉树 + LOD 自适应显示
- **滤波处理**：多种点云滤波与降采样算法
- **配准**：支持粗配准和精配准的完整配准工作流
- **分割**：形状检测、区域生长、聚类等多种分割方法
- **地面分割**：基于 CSF 的布料模拟滤波
- **植被分割**：基于高程归一化与特征分类的植被提取
- **变化检测**：云对云、云对网格距离计算
- **曲面重建**：点云到网格的转换
- **测量**：距离、角度、面积等测量工具
- **Python 脚本**：内嵌 Python 解释器，支持自动化批处理

```
└─────────────────────────────────────┘
```

## 适用场景

- 测绘与遥感数据处理
- 三维建模与重建
- 变形监测与变化检测
- 地形分析与地面滤波
- 点云数据预处理与质量控制
