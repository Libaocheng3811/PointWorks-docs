# 五分钟上手

本教程将带你快速体验 PointWorks 的核心功能。

## 1. 打开点云文件

通过以下任一方式加载点云：

- 菜单栏：`文件` → `打开`
- 工具栏：点击打开按钮
- 拖拽文件到主窗口

支持 LAS、LAZ、PLY、PCD、E57、TXT 等格式。

## 2. 浏览点云

| 操作 | 鼠标 | 快捷键 |
|------|------|--------|
| 旋转 | 左键拖拽 | — |
| 平移 | 中键拖拽 | — |
| 缩放 | 滚轮 | — |
| 聚焦选中点云 | — | `F` 键 |
| 重置视图 | — | `R` 键 |

## 3. 滤波降采样

1. 在文件树中选中点云
2. 菜单栏：`滤波` → `体素降采样`
3. 设置体素大小（如 0.05m）
4. 点击确定执行

## 4. 地面分割

1. 选中点云
2. 菜单栏：`地面分割` → `CSF 布料模拟`
3. 调整参数（布料分辨率、迭代次数等）
4. 执行后得到地面点和非地面点

## 5. 导出结果

1. 在文件树中选中要导出的点云
2. 菜单栏：`文件` → `另存为`
3. 选择输出格式和路径
4. 点击保存
