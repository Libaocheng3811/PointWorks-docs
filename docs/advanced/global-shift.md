# 全局坐标偏移

## 问题背景

测绘数据通常使用大地坐标（如 CGCS2000），坐标值可能达到数百万（如 X: 3500000, Y: 500000）。当坐标值超过约 10^5 时，32 位浮点数的精度不足以表示亚毫米级别，导致显示和计算出现精度丢失。

## 解决方案

PointWorks 在加载点云时自动检测并应用全局坐标偏移（Global Shift）：

1. 计算点云的重心坐标
2. 将所有点减去重心坐标，转换为局部坐标
3. 记录偏移量用于坐标还原

## 操作方式

### 自动检测

加载文件时，PointWorks 自动检测大坐标值并提示应用偏移。

### 手动设置

菜单栏：`编辑` → `全局偏移`

| 参数 | 说明 |
|------|------|
| X 偏移 | X 方向偏移量 |
| Y 偏移 | Y 方向偏移量 |
| Z 偏移 | Z 方向偏移量 |

## 偏移量的保存与传递

- 偏移量存储在点云对象的元数据中
- 导出文件时自动写入偏移信息（LAS 文件的 VLR 中）
- 配准、变化检测等操作会自动处理偏移量

!!! tip "提示"
    如果导入后发现点云坐标异常（如全部聚集在一个点），检查是否未正确应用全局偏移。
