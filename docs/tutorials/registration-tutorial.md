# 点云配准实战

本教程介绍如何使用 PointWorks 将两站或多站激光扫描点云配准到统一坐标系中。

## 概述

点云配准（Registration）是将不同视角、不同时间获取的多份点云对齐到统一坐标系的过程。典型应用包括：

- 多站地面激光扫描拼接
- 机载与地面 LiDAR 数据融合
- 变化检测的预处理
- 建筑物 / 场景完整三维重建

PointWorks 提供完整的配准工作流：

```
粗配准（初始对齐） → 精配准（ICP 优化） → 结果验证
```

## 配准策略

| 阶段 | 工具 | 说明 |
|------|------|------|
| 初始对齐 | Center Alignment | 两点云质心对齐 |
| 粗配准 | Global Registration | 基于特征的自动粗配准 |
| 手动配准 | Point Pairs Alignment | 手动选取对应点对 |
| 精配准 | Fine Registration (ICP) | 迭代最近点精细对齐 |

## 第一步：加载两站点云

1. **File > Open** 加载第一站点云（源点云）
2. 再次 **File > Open** 加载第二站点云（目标点云）
3. 两站点云同时显示在 3D 视图中，初始位置可能完全分离

:::tip[点云颜色区分]
使用 **Edit > Color** 为两站点云设置不同颜色，便于区分和对齐判断。

:::
## 第二步：粗配准

粗配准的目的是将两站点云大致对齐，为后续精配准提供良好的初始位置。

### 方法一：中心对齐

如果两站点云仅存在平移关系，可使用中心对齐：

1. **Tools > Align > Align by Centers**
2. 选择源点云和目标点云
3. 点击 **Apply**，两点云质心将对齐到同一位置

### 方法二：点对配准（推荐）

适用于有重叠区域且可辨识对应特征的情况：

1. **Tools > Align > Point Pairs Alignment**
2. 在源点云上依次拾取 3 个以上对应点
3. 在目标点云上依次拾取对应的 3 个以上同名点
4. 点对顺序必须一致
5. 点击 **Apply** 计算变换矩阵并应用

:::warning[点对选取原则]
- 至少选择 3 对不共线的点
- 点对应均匀分布在重叠区域
- 选择特征明显的点（角点、边缘点等）
- 对应点对的选取顺序必须一致

:::
### 方法三：自动全局配准

基于特征描述符的自动粗配准：

1. **Tools > Align > Global Registration**
2. 选择配准算法：

| 算法 | 说明 |
|------|------|
| IA-RANSAC | 基于采样一致性的初始对齐 |
| SCPR | 基于预剔除的采样一致性 |
| FPCS | 4点一致集算法 |
| NDT | 正态分布变换 |

3. 选择特征描述符（FPFH 推荐）
4. 设置搜索半径和 RANSAC 参数
5. 点击 **Apply** 执行配准

## 第三步：精配准（ICP）

粗配准后，使用 ICP 算法进行精细对齐。

### 1. 打开精配准工具

**Tools > Align > Fine Registration**

### 2. 选择 ICP 算法

| 算法 | 适用场景 |
|------|----------|
| **ICP** | 通用点对点配准 |
| **ICPWithNormals** | 有法线信息时，精度更高 |
| **ICPNonLinear** | 非线性优化，适合大变形 |

### 3. 配置参数

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| **MaxCorrespondenceDistance** | 1.0 ~ 5.0 | 最大对应点距离阈值，超过此距离的点对不参与配准 |
| **MaximumIterations** | 50 ~ 100 | 最大迭代次数 |
| **TransformationEpsilon** | 1e-8 | 变换矩阵收敛阈值 |
| **EuclideanFitnessEpsilon** | 1e-6 | 欧式适应度收敛阈值 |

:::tip[参数调优]
- `MaxCorrespondenceDistance` 应从大值开始，逐步缩小
- 第一次 ICP 用较大阈值（如 5.0），第二次用较小阈值（如 1.0）
- 如果点云有法线，优先使用 ICPWithNormals

:::
### 4. 执行精配准

1. 确认源点云和目标点云选择正确
2. 点击 **Apply**
3. 进度条显示迭代过程
4. 完成后在控制台查看配准结果（适应度分数、变换矩阵）

## 第四步：结果验证

### 视觉检查

- 旋转 3D 视图，从不同角度检查重叠区域是否对齐
- 使用不同颜色渲染两站点云，查看颜色混合是否均匀
- 放大检查细节区域（建筑边缘、地面纹理等）

### 数值验证

配准完成后控制台输出：

- **Fitness Score** - 适应度分数（越小越好，一般 < 0.01 为优）
- **Transformation Matrix** - 4x4 变换矩阵（平移 + 旋转）

### 配准质量判断标准

| 适应度分数 | 配准质量 |
|------------|----------|
| < 0.001 | 优秀 |
| 0.001 ~ 0.01 | 良好 |
| 0.01 ~ 0.05 | 可接受 |
| > 0.05 | 需要重新配准 |

## 多站点云配准

对于多站扫描数据：

1. 选择一个站点作为参考站（不移动）
2. 依次将相邻站点与参考站配准
3. 使用已配准的站点作为新参考站，继续配准下一站
4. 逐站累积配准，直到所有站点对齐

:::warning[累积误差]
多站配准会累积误差，站点越多末端偏差越大。建议使用闭合环扫描（首尾站有重叠），可有效控制累积误差。

:::
## 导出配准结果

1. 在点云树中选中配准后的点云
2. 右键 **Save As** 导出
3. 建议使用 LAS 格式，保留坐标信息

如果需要合并多站点云：
1. 选中多个已配准的点云节点
2. 右键选择 **Merge** 合并为单一点云
3. 导出合并结果
