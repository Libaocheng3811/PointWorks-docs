# 变化检测实战

本教程介绍如何使用 PointWorks 的变化检测插件对两期（或多期）点云数据进行对比分析。

## 概述

点云变化检测通过计算两份不同时期采集的点云之间的距离差异，识别场景中的变化区域。典型应用包括：

- 建筑施工监测
- 地形变化分析（滑坡、沉降）
- 植被生长监测
- 城市更新追踪
- 安全巡检（非法建筑、道路损毁）

## 前提条件

在进行变化检测之前，两期点云必须满足以下条件：

1. **已配准到同一坐标系** - 参见 [点云配准实战](registration-tutorial.md)
2. **覆盖相同或大部分重叠区域**
3. **点云密度合理** - 差异过大可能影响检测结果

!!! danger "配准是关键"
    配准误差是变化检测最大的误差来源。如果两期数据未精确配准，会产生大量虚假变化。务必先完成精配准（ICP 适应度分数 < 0.01）。

## 第一步：加载两期数据

1. **File > Open** 加载第一时期（T1）点云
2. **File > Open** 加载第二时期（T2）点云
3. 确认两期数据已在同一坐标系下正确对齐

## 第二步：使用变化检测插件

### 1. 打开变化检测插件

通过菜单 **Plugins > Change Detection** 打开变化检测工具浮窗。

### 2. 选择输入点云

- **Reference Cloud** - 参考点云（T1 时期）
- **Compare Cloud** - 对比点云（T2 时期）

在点云树中分别选中对应的点云节点。

### 3. 选择距离计算方法

PointWorks 提供以下距离计算方法：

| 方法 | 说明 | 适用场景 |
|------|------|----------|
| **C2C Nearest** | 最近邻距离 | 通用场景，速度最快 |
| **C2C KNN Mean** | K 近邻平均距离 | 减少噪声影响 |
| **C2C Radius Mean** | 半径内平均距离 | 密度不均匀时 |
| **M3C2** | 多尺度模型到模型云比较 | 精确变化检测（预留） |

!!! tip "方法选择"
    - 首次分析建议使用 **C2C Nearest**，速度快且直观
    - 对精度要求高时使用 **C2C KNN Mean**（K=5~10）
    - 点云密度差异大时使用 **C2C Radius Mean**

### 4. 配置参数

**C2C Nearest 参数**：

| 参数 | 说明 |
|------|------|
| **Max Distance** | 最大距离阈值，超过此值的距离被截断 |

**C2C KNN Mean 参数**：

| 参数 | 说明 |
|------|------|
| **K** | 最近邻数量 |
| **Max Distance** | 最大距离阈值 |

**C2C Radius Mean 参数**：

| 参数 | 说明 |
|------|------|
| **Radius** | 搜索半径 |
| **Max Distance** | 最大距离阈值 |

### 5. 执行变化检测

1. 点击 **Apply** 按钮
2. 进度对话框显示处理进度
3. 处理时间取决于点云大小和所选方法

### 6. 查看结果

处理完成后：

- 结果点云使用 **Jet 色带**（蓝-绿-黄-红）渲染距离值
- 蓝色表示距离小（无变化或微小变化）
- 红色表示距离大（显著变化）
- 在点云树中右键结果点云，可查看标量场统计信息

## 结果解读

### 距离色带说明

| 颜色 | 距离范围 | 含义 |
|------|----------|------|
| 深蓝 | 接近 0 | 无变化区域 |
| 青/绿 | 较小 | 可能是噪声或微小变化 |
| 黄/橙 | 中等 | 明显变化 |
| 红 | 较大 | 显著变化 |

### 常见变化类型

| 变化类型 | 距离特征 | 典型应用 |
|----------|----------|----------|
| 新建建筑 | T1 无，T2 有，单向正距离 | 城市监测 |
| 拆除建筑 | T1 有，T2 无，需反向检测 | 拆违监控 |
| 地面沉降 | 均匀的微小负距离 | 地质监测 |
| 植被生长 | 不规则正距离 | 生态监测 |
| 车辆移动 | 局部小范围变化 | 交通监控 |

!!! warning "双向变化检测"
    C2C Nearest 默认计算 T2 到 T1 的单向距离。拆除类变化需要交换参考点云再次检测，或使用 M3C2 方法（支持双向检测）。

## 进阶分析

### 变化区域提取

检测完成后，可以通过滤波提取显著变化区域：

1. 选中变化检测结果点云
2. **Tools > Filters > PassThrough**
3. 对距离标量场设置阈值（如 > 0.5m 为显著变化）
4. 提取出显著变化区域

### 变化体积估算

对于沉降或堆填场景：

1. 提取变化区域的点云
2. 使用距离标量场乘以点间距估算面积
3. 距离值乘以面积可近似体积变化

### 批量多期检测

使用 Python 脚本进行多期数据批量变化检测，参见 [Python 自动化实战](python-automation.md)。

## 导出检测结果

1. 在点云树中选中检测结果点云
2. 右键 **Save As**
3. 推荐使用 LAS 格式导出，距离值作为标量场保存
4. 在 GIS 软件（如 CloudCompare、QGIS）中进一步分析

## 常见问题

### 检测结果全是红色（变化很大）？

- 检查两期数据是否已正确配准
- 重新执行 ICP 精配准
- 确认坐标系一致

### 检测结果几乎没有变化？

- 增大最大距离阈值
- 检查参考/对比点云是否选反了
- 确认两期数据确实存在时间差异

### 结果中有大量细碎噪声？

- 使用 C2C KNN Mean 方法（K=5~10）
- 对原始点云先做统计离群点滤波
- 增大距离阈值过滤微小变化
