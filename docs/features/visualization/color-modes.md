# 颜色显示模式

PointWorks 提供多种点云着色方式，帮助你从不同角度理解和分析数据特征。颜色模式通过菜单栏 `编辑 > 颜色` 进行切换。

## 可用颜色模式

### RGB 原色模式

显示点云采集时记录的真实颜色信息。这是最直观的显示模式，适用于摄影测量点云和带颜色的激光扫描数据。

| 属性 | 说明 |
|------|------|
| 数据来源 | 点云自带的 RGB 颜色字段 |
| 适用格式 | LAS（点格式 2/3/7/8）、PLY、PCD、TXT（含 RGB 列） |
| 渲染方式 | 直接将 R/G/B 值映射到显示颜色 |

!!! warning "无颜色数据"
    如果点云文件不包含颜色信息（如 LAS 点格式 0/1），RGB 模式下所有点将显示为白色或默认颜色。此时建议切换到其他着色模式。

### 高度着色模式

根据点的 Z 坐标（高程值）映射颜色，使用色带从低到高渐变。适用于地形、建筑物等场景的高程分析。

| 设置项 | 说明 |
|--------|------|
| 色带类型 | 可选多种色带（Jet、Rainbow、Terrain 等） |
| 值域范围 | 自动计算或手动设置最小/最大高程 |
| 高程方向 | 默认使用 Z 轴，可切换为 X 或 Y 轴 |

### 强度着色模式

根据激光雷达回波强度值着色。不同材质对激光的反射强度不同，可用于初步区分地物类型。

| 设置项 | 说明 |
|--------|------|
| 数据来源 | LAS 文件的 Intensity 字段 |
| 色带类型 | 灰度或自定义色带 |
| 值域范围 | 自动或手动设置 |

!!! info "强度数据"
    强度值仅存在于激光雷达点云中（LAS/LAZ 格式）。摄影测量点云通常不包含强度信息。

### 分类着色模式

按 LAS 标准分类码着色，不同类别使用不同颜色，便于直观查看分类结果。

| 分类码 | 颜色（默认） | 说明 |
|--------|-------------|------|
| 0 | 灰色 | 未分类 |
| 1 | 白色 | 未分类 |
| 2 | 棕色 | 地面 |
| 3 | 浅绿 | 低植被 |
| 4 | 绿色 | 中植被 |
| 5 | 深绿 | 高植被 |
| 6 | 黄色 | 建筑物 |
| 7 | 紫色 | 低噪声 |
| 8 | 红色 | 高噪声/保留 |
| 9 | 蓝色 | 水体 |

### 标量场着色模式

根据点云的自定义标量场数据着色。标量场可以是算法计算的结果（如距离、曲率等），也可以是外部导入的属性数据。

| 设置项 | 说明 |
|--------|------|
| 标量字段 | 选择要显示的标量字段名称 |
| 色带类型 | 可选多种色带 |
| 值域范围 | 自动计算或手动设置 |
| 色度条 | 在视图中显示色度条及数值范围 |

!!! tip "标量场"
    许多算法（如距离计算、法线估计）的结果会自动生成为标量场。你可以在颜色设置中选择对应的标量字段来可视化结果。

### 法线方向着色模式

根据点的法线方向映射颜色。法线的 X/Y/Z 分量分别映射到 R/G/B 通道，形成方向编码颜色。

| 条件 | 说明 |
|------|------|
| 前提条件 | 点云已计算法线 |
| 计算方法 | 菜单栏 `编辑 > 法线 > 计算法线` |
| 显示效果 | 朝向相似的面呈现相似颜色 |

## 色带选择

PointWorks 内置多种色带，适用于不同的可视化需求：

| 色带 | 适用场景 |
|------|---------|
| Jet | 通用科学可视化 |
| Rainbow | 高度/温度等连续值 |
| Gray | 强度数据、单变量 |
| Terrain | 地形高程 |
| Hot | 热力图、密度 |
| Cool/Warm | 对称数据（正负值） |

## 切换颜色模式

### 操作步骤

1. 在左侧树视图中选中目标点云
2. 菜单栏：`编辑 > 颜色` 打开颜色设置面板
3. 在颜色设置面板中选择所需的着色模式
4. 根据模式配置相应参数（色带、值域范围等）
5. 实时预览效果

### 快速切换

| 操作 | 方法 |
|------|------|
| RGB 模式 | 颜色面板中选择 "RGB" |
| 高度着色 | 颜色面板中选择 "高度" |
| 单色统一 | 颜色面板中选择 "纯色" 并选取颜色 |
