# 三维视图操作

PointWorks 的三维视图提供了丰富的鼠标和键盘交互方式，方便你从任意角度观察点云数据。

## 鼠标操作

### 基本操作

| 操作 | 鼠标 | 说明 |
|------|------|------|
| 旋转 | 左键拖动 | 绕场景中心旋转相机 |
| 平移 | 中键拖动（或 Shift + 左键） | 在屏幕平面内移动视图 |
| 缩放 | 滚轮 | 放大或缩小视图 |
| 框选缩放 | 右键拖动 | 框选区域并放大至充满视图 |

### 高级操作

| 操作 | 鼠标 | 说明 |
|------|------|------|
| 点拾取 | 左键单击 | 选中单个点，显示其坐标和属性信息 |
| 多边形选择 | 按住 Ctrl + 左键依次点击 | 绘制多边形区域选择多个点 |
| 重置视图 | 双击中键 | 重置相机到默认视角 |

!!! tip "旋转中心"
    旋转操作默认绕场景中心进行。使用"聚焦"功能可以将旋转中心设置到指定位置，便于局部观察。

## 键盘快捷键

### 视图控制

| 快捷键 | 功能 |
|--------|------|
| `R` | 重置视图到默认位置 |
| `F` | 聚焦选中的点云（Fit to View） |
| `C` | 切换坐标轴显示 |
| `+` / `=` | 增大点大小 |
| `-` | 减小点大小 |

### 视角预设

| 快捷键 | 功能 |
|--------|------|
| `1` | 正视图（Front） -- 从 Y 轴正方向看 |
| `2` | 后视图（Back） -- 从 Y 轴负方向看 |
| `3` | 左视图（Left） -- 从 X 轴负方向看 |
| `4` | 右视图（Right） -- 从 X 轴正方向看 |
| `5` | 顶视图（Top） -- 从 Z 轴正方向看 |
| `6` | 底视图（Bottom） -- 从 Z 轴负方向看 |
| `0` | 等轴测视图（Isometric） |

## 聚焦功能

聚焦功能将视图相机移动到指定点云的中心，并调整缩放比例使目标完全可见。

### 使用方法

1. 在左侧树视图中选中目标点云
2. 按快捷键 `F`，或右键选择 `聚焦`
3. 视图将自动平移和缩放，使选中的点云居中显示

### 应用场景

| 场景 | 说明 |
|------|------|
| 多点云定位 | 加载多个点云后，快速定位到某一个 |
| 局部观察 | 在大场景中聚焦到感兴趣的区域 |
| 操作后查看 | 滤波或分割后查看结果点云 |

## 点拾取

点拾取功能允许你点击三维视图中的任意点，查看其详细信息。

### 使用方法

1. 确保拾取模式已启用（默认启用）
2. 在三维视图中左键单击某个点
3. 状态栏或信息面板中会显示该点的详细信息

### 显示信息

| 信息 | 说明 |
|------|------|
| 坐标 (X, Y, Z) | 点的三维空间坐标 |
| 颜色 (R, G, B) | 点的颜色值 |
| 强度 | 点的强度值（如果存在） |
| 分类 | LAS 分类码（如果存在） |
| 法线 (NX, NY, NZ) | 法线方向（如果已计算） |

!!! tip "精确拾取"
    在密集点云中拾取特定点时，可以先放大视图到该区域，然后适当增大点大小，这样更容易精确选中目标点。

## 多点云视图管理

当工作区中加载了多个点云时：

| 操作 | 方法 |
|------|------|
| 显示/隐藏单个点云 | 在树视图中勾选/取消勾选节点 |
| 显示全部 | 菜单栏 `视图 > 显示全部` |
| 隐藏全部 | 菜单栏 `视图 > 隐藏全部` |
| 仅显示选中 | 右键菜单 `仅显示此项` |
