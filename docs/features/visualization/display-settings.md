# 显示设置

显示设置允许你自定义三维视图的全局显示参数，包括点大小、背景颜色、包围盒颜色等。这些设置影响整体视觉效果，不改变数据本身。

## 打开显示设置

菜单栏：`选项 > 显示设置`（Options > Display Settings）

显示设置对话框采用左侧导航列表 + 右侧设置页面的布局结构，便于分类管理各项设置。

## 设置项一览

### 点大小

控制三维视图中点云的渲染点大小（以像素为单位）。

| 设置项 | 说明 |
|--------|------|
| 点大小范围 | 1 - 20 像素 |
| 默认值 | 2 像素 |
| 调整方式 | 滑块或输入框 |
| 快捷调整 | 键盘 `+` 增大，`-` 减小 |

:::tip[点大小建议]
- 密集点云使用较小的点（1-3），避免遮挡细节
- 稀疏点云使用较大的点（4-8），便于观察分布
- 演示展示时适当增大点大小，提升视觉效果

:::
### 背景颜色

设置三维视图的背景颜色。

| 选项 | 说明 |
|------|------|
| 预设颜色 | 黑色、深灰、白色、浅灰 |
| 自定义颜色 | 通过颜色选择器自定义任意颜色 |
| 默认值 | 深灰 (#333333) |

**预设方案说明**：

| 预设 | 颜色 | 适用场景 |
|------|------|---------|
| 黑色 | #000000 | 彩色点云对比鲜明 |
| 深灰 | #333333 | 默认，适合大多数场景 |
| 白色 | #FFFFFF | 截图用于论文报告 |
| 浅灰 | #CCCCCC | 柔和观感 |

### 包围盒颜色

设置选中点云的包围盒（Bounding Box）线框显示颜色。

| 设置项 | 说明 |
|--------|------|
| 线框颜色 | 包围盒边框颜色 |
| 默认值 | 黄色 (#FFFF00) |
| 显示/隐藏 | 控制包围盒是否可见 |

## 设置页面管理

显示设置对话框采用可扩展的页面架构：

```
┌──────────────────────────────────┐
│  ┌─────────┐  ┌───────────────┐  │
│  │ 颜色    │  │               │  │
│  │ ...     │  │  设置内容区   │  │
│  │ ...     │  │               │  │
│  └─────────┘  └───────────────┘  │
│                                  │
│  [ 应用 ]  [ 重置 ]  [ 关闭 ]    │
└──────────────────────────────────┘
```

| 按钮 | 功能 |
|------|------|
| 应用 | 将当前设置应用到视图 |
| 重置 | 恢复到默认设置 |
| 关闭 | 关闭设置对话框 |

:::info[即时预览]
部分设置（如点大小、背景色）在调整时会实时预览效果，无需点击应用按钮。

:::
## 视图工具栏

除了显示设置对话框外，三维视图的工具栏也提供快捷的显示控制：

| 工具 | 功能 |
|------|------|
| 点大小滑块 | 快速调整当前点大小 |
| 重置视图 | 重置相机到默认位置 |
| 坐标轴开关 | 显示/隐藏坐标轴 |
| 截图 | 导出当前视图为图片 |
