# 可视化

可视化模块基于 VTK 渲染引擎，提供高性能的三维点云与网格显示能力。该模块由 `ct_viz` 库实现，核心组件包括 `CloudView` 三维视图控件和 `OctreeRenderer` 高级渲染器。

## 模块组成

### [三维视图操作](view-controls)

在三维视图中对点云进行交互式查看，包括旋转、平移、缩放、聚焦等操作。

### [颜色显示模式](color-modes)

多种点云着色方式，帮助从不同角度理解数据特征：

- RGB 原色模式 -- 显示点云采集时的真实颜色
- 标量场模式 -- 按高度、强度等标量值着色
- 分类着色 -- 按 LAS 分类码着色
- 法线方向着色 -- 按法线方向映射颜色

### [显示设置](display-settings)

全局显示参数配置，包括点大小、背景颜色、包围盒颜色等。

## 渲染架构

PointWorks 采用高性能渲染管线，针对大规模点云（千万级以上）做了深度优化：

```
数据存储 (CloudBlock/AOS)
      |
      v
OctreeRenderer (SSE 遍历 + 视锥剔除)
      |
      v
VTK Actor 对象池 (最多 500 个)
      |
      v
GPU 渲染
```

### 核心技术

| 技术 | 说明 |
|------|------|
| 八叉树空间索引 | 将点云按空间递归划分为 8 个子区域，叶子节点最多 6 万点 |
| SSE (Screen Space Error) | 根据屏幕投影大小决定渲染精细度 |
| 视锥剔除 | 仅渲染当前相机视野内的数据块 |
| 动态 LOD | 交互时降低精度保证流畅，静止时恢复精细渲染 |
| Actor 对象池 | 复用 VTK Actor 对象，避免频繁创建销毁 |
| 全局坐标偏移 | 自动将大坐标平移至原点附近，解决 GPU 精度问题 |

### 性能指标

| 点云规模 | 渲染策略 |
|----------|---------|
| < 1000 万点 | 直通模式，直接渲染全部点 |
| >= 1000 万点 | 八叉树模式，启用 LOD 和视锥剔除 |

## 交互功能

三维视图还支持以下交互功能：

| 功能 | 说明 |
|------|------|
| 点拾取 | 点击查看单个点的坐标、颜色等属性信息 |
| 多边形选择 | 绘制多边形区域选择一组点 |
| 截图 | 将当前视图导出为图片 |
| 坐标轴显示 | 显示/隐藏三维坐标轴参考 |

:::tip[大点云交互]
对于超大点云，在旋转和缩放操作时渲染器会自动降低显示精度以保持流畅。松开鼠标后，渲染器会在短时间内恢复到最佳显示效果。

:::
