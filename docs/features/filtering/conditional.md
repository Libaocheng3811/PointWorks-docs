# 条件滤波 (Conditional Removal)

条件滤波允许用户通过构造逻辑条件表达式来精确筛选点云中的点。用户可以对点云的任意字段（如坐标 X/Y/Z、强度 Intensity、曲率 Curvature 等）设置一个或多个比较条件，并使用逻辑运算符组合这些条件。满足条件的点被保留，不满足的将被移除。该方法基于 PCL 的 `ConditionalRemoval` 滤波器实现。

## 入口路径

`滤波 > 条件滤波 (Conditional Removal)`

## 参数说明

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| 字段名称 (Field) | string | z | 用于条件判断的字段名，如 `x`、`y`、`z`、`intensity`、`curvature` 等 |
| 比较运算符 | enum | GT | 比较操作类型，可选值见下表 |
| 阈值 (Value) | float | 0.0 | 用于比较的阈值 |
| 条件组合方式 | enum | AND | 多个条件之间的逻辑关系：`AND`（全部满足）或 `OR`（满足任一） |
| 反转 (Negative) | bool | false | 勾选后保留不满足条件的点 |

### 比较运算符

| 运算符 | 含义 | 等价数学表达 |
|--------|------|-------------|
| \`GT\` | 大于 | field {'>'} value |
| \`GE\` | 大于等于 | field {'>='} value |
| \`LT\` | 小于 | field {'<'} value |
| \`LE\` | 小于等于 | field {'<='} value |
| \`EQ\` | 等于 | field {'=='} value |
| \`NE\` | 不等于 | field {'!='} value |

## 操作步骤

1. 在点云列表中选中目标点云对象
2. 点击菜单 `滤波 > 条件滤波 (Conditional Removal)`
3. 在参数对话框中添加一个或多个条件：
   - 选择**字段名称**
   - 选择**比较运算符**
   - 输入**阈值**
4. 如有多个条件，选择**条件组合方式**（AND / OR）
5. 点击**确定**执行滤波
6. 滤波结果作为新点云对象添加到列表中

## 条件表达式示例

| 需求 | 条件设置 |
|------|---------|
| 仅保留高程在 0.5m 到 2.0m 之间的点 | 条件1: `z GE 0.5`，条件2: `z LE 2.0`，组合: AND |
| 移除强度值低于 10 的点 | 条件: `intensity GE 10` |
| 保留 X > 5 或 Y > 5 的点 | 条件1: `x GT 5`，条件2: `y GT 5`，组合: OR |
| 移除曲率为 0 的平坦区域 | 条件: `curvature GT 0` |

## 适用场景

- **基于属性的精确筛选**：根据点云中任意可用字段进行灵活过滤
- **强度值过滤**：去除低强度（低信噪比）的不可靠点
- **曲率过滤**：保留曲率较大的特征区域（如边缘、角点）
- **多条件组合裁剪**：当单一维度的直通滤波无法满足需求时

:::note[提示]
可用字段取决于点云包含的数据。标准 XYZ 点云仅支持 `x`、`y`、`z` 三个字段。如果点云经过法线估计，还可以使用 `curvature` 字段；如果点云包含强度信息，可以使用 `intensity` 字段。可在点云属性面板中查看当前点云包含的字段列表。

:::
:::warning[注意]
对 `EQ`（等于）运算符使用浮点数值时需格外小心。由于浮点精度问题，精确匹配可能无法命中任何点。建议使用 `GE` / `LE` 组合或一个较小的范围替代精确匹配。


:::
