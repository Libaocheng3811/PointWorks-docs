# 半径离群点移除 (Radius Outlier Removal)

半径离群点移除是一种基于局部邻域密度的离群点检测方法。对于点云中的每个点，算法统计以该点为圆心、指定搜索半径范围内的邻居数量。如果邻居数量低于设定的最小阈值，该点即被判定为离群点并移除。该方法基于 PCL 的 `RadiusOutlierRemoval` 滤波器实现。

相比统计离群点移除，半径离群点移除更加直观：它直接检查每个点在其周围一定范围内是否有足够的邻居，适合处理密度分布不均匀的点云。

## 入口路径

`滤波 > 半径离群点移除 (Radius Outlier Removal)`

## 参数说明

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| 搜索半径 (Radius) | float | 0.1 | 用于搜索邻居的球形区域半径（单位：米） |
| 最小邻居数 (MinNeighbors) | int | 5 | 半径范围内必须包含的最少邻居点数。少于此数的点被视为离群点 |
| 反转 (Negative) | bool | false | 勾选后保留离群点，移除内点 |

## 操作步骤

1. 在点云列表中选中目标点云对象
2. 点击菜单 `滤波 > 半径离群点移除 (Radius Outlier Removal)`
3. 设置**搜索半径**和**最小邻居数**
4. 点击**确定**执行滤波
5. 滤波结果作为新点云对象添加到列表中

## 参数调优建议

- **搜索半径**应与点云的平均点间距匹配：
  - 对于地面激光扫描点云，通常 `0.1 ~ 0.5` 米
  - 对于近距离精细扫描，通常 `0.005 ~ 0.02` 米
  - 对于机载激光雷达点云，可能需要 `1.0 ~ 5.0` 米
- **最小邻居数**决定过滤的严格程度：
  - `2 ~ 5`：宽松过滤，仅去除极端孤立点
  - `10 ~ 30`：中等过滤，适合大多数场景
  - `50` 以上：严格过滤，需要根据点云密度调整

## 适用场景

- **飘散点去除**：移除远离主体点云的孤立噪声点（如空气中的灰尘反射、多路径效应产生的假点）
- **密度不一致场景**：当点云各区域密度差异较大，统计离群点移除效果不理想时，半径方法能更好地适应
- **建筑物立面清洗**：去除建筑扫描中窗户反射等产生的零散点

:::note[提示]
搜索半径的选择可以参考点云的平均点间距。在三维视窗中使用"测距"工具测量若干相邻点之间的距离，取平均值后再乘以 3 ~ 5 倍作为搜索半径的初始值。

:::
:::warning[注意]
如果搜索半径设置过小而最小邻居数设置过大，可能导致边缘区域的正常点被误删（因为边缘点的邻域在单侧天然较少）。建议对包含明显边界的点云适当放宽参数，或在滤波后检查边缘完整性。


:::
