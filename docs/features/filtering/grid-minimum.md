# 网格最小值 (Grid Minimum)

网格最小值滤波是一种基于二维网格的降采样方法。算法将点云投影到 X-Y 平面上，将其划分为等大的正方形网格，然后在每个网格内仅保留 Z 值（高程）最小的那个点。这种方法能够有效提取点云中的最低表面，常用于地面点提取和数字高程模型（DEM）生成。

## 入口路径

`滤波 > 网格最小值 (Grid Minimum)`

## 参数说明

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| 网格大小 (Grid Size) | float | 1.0 | X-Y 平面上正方形网格的边长（单位：米）。值越大，输出点越稀疏，表面越平滑 |

## 操作步骤

1. 在点云列表中选中目标点云对象
2. 点击菜单 `滤波 > 网格最小值 (Grid Minimum)`
3. 在参数对话框中设置**网格大小**
4. 点击**确定**执行滤波
5. 滤波结果作为新点云对象添加到列表中

## 参数调优建议

- **网格大小**的选择取决于应用场景和点云密度：
  - 精细地面提取：`0.1 ~ 0.5` 米
  - 常规 DEM 生成：`0.5 ~ 2.0` 米
  - 大范围地形简化：`2.0 ~ 10.0` 米
- 网格大小应不小于点云的平均点间距，否则部分网格内可能没有点

## 适用场景

- **地面点提取**：从包含建筑物、植被等的复杂场景中提取地面最低点
- **DEM/DTM 生成**：生成数字高程模型或数字地形模型
- **地面平整度检测**：提取地面点后分析其平整度
- **隧道底板提取**：从隧道扫描点云中提取底板最低表面

!!! note "提示"
    网格最小值滤波假设最低点代表地面。在存在地下结构（如地下室入口、坑洞）的场景中，可能需要先进行直通滤波去除异常低点，再执行网格最小值操作。

!!! tip "技巧"
    可以先用网格最小值提取地面，再用原始点云减去地面点云（或使用反转的直通滤波），即可分离出非地面点（建筑物、植被等），这是一种简单有效的地面/非地面分割方法。
