# 滤波

滤波模块提供多种点云降采样和噪声去除算法，帮助你在保留数据特征的前提下减少数据量或提升数据质量。所有滤波算法位于 `libs/algorithm/filters.h`，通过菜单栏 `滤波` 访问。

## 概述

点云采集过程中，受传感器精度、环境干扰等因素影响，原始数据通常包含噪声点和冗余数据。滤波是点云预处理的关键步骤，直接影响后续算法的效果和性能。

## 支持的滤波器

### 体素降采样 (VoxelGrid)

将三维空间划分为固定大小的体素网格，每个体素内的所有点用其质心点代替，实现均匀降采样。

| 参数 | 类型 | 说明 |
|------|------|------|
| 体素大小 X | float | X 方向体素边长（米） |
| 体素大小 Y | float | Y 方向体素边长（米） |
| 体素大小 Z | float | Z 方向体素边长（米） |

**适用场景**：

- 统一点云密度
- 大规模点云快速降采样
- 在保留空间分布特征的前提下减少数据量

:::tip[体素大小选择]
体素大小取决于点云的实际需求和原始密度。一般设置为平均点间距的 2-5 倍，可在保留整体形状的同时显著减少点数。

:::
### 统计离群点移除 (StatisticalOutlierRemoval)

对每个点计算其到邻近点的平均距离，假设结果服从高斯分布，剔除超出指定标准差倍数的点。

| 参数 | 类型 | 说明 |
|------|------|------|
| 邻域点数 K | int | 用于计算平均距离的最近邻点数 |
| 标准差倍数 | float | 判定阈值（乘以标准差） |

**适用场景**：

- 移除稀疏离群噪声点
- 点云去噪
- 传感器误差导致的异常点清除

**参数建议**：

| 点云密度 | 邻域点数 K | 标准差倍数 |
|----------|-----------|-----------|
| 密集（> 100 点/m^2） | 20-50 | 1.0-2.0 |
| 中等 | 10-20 | 2.0-3.0 |
| 稀疏 | 5-10 | 3.0-5.0 |

### 半径离群点移除 (RadiusOutlierRemoval)

对每个点在指定半径内搜索邻域点，如果邻域点数少于阈值则判定为离群点并移除。

| 参数 | 类型 | 说明 |
|------|------|------|
| 搜索半径 | float | 邻域搜索半径（米） |
| 最小邻域点数 | int | 半径内至少需要的邻域点数 |

**适用场景**：

- 移除空间上孤立的噪声点
- 清除点云边缘的飞散点
- 与统计离群点移除互补使用

### 直通滤波 (PassThrough)

沿指定坐标轴设置范围阈值，保留范围内的点，范围外的点被滤除。

| 参数 | 类型 | 说明 |
|------|------|------|
| 滤波轴 | enum | X / Y / Z 轴 |
| 最小值 | float | 保留范围下限 |
| 最大值 | float | 保留范围上限 |
| 取反 | bool | 是否反转（保留范围外的点） |

**适用场景**：

- 按高程范围裁剪（如只保留地面附近点）
- 按平面范围限定研究区域
- 快速粗略裁剪

:::tip[取反功能]
勾选"取反"可以保留范围外的点，这在排除特定区域时很有用。例如，排除已知的高程异常区域。

:::
### 条件滤波 (ConditionalRemoval)

根据用户定义的条件对点云进行布尔筛选，仅保留满足条件的点。

| 参数 | 类型 | 说明 |
|------|------|------|
| 条件表达式 | string | 基于点属性的布尔表达式 |

**适用场景**：

- 多条件组合筛选
- 基于强度、颜色等属性的精确过滤

### 2D 网格最小值投影 (GridMinimum)

将点云投影到二维平面网格，每个网格单元仅保留高程最低的点。

| 参数 | 类型 | 说明 |
|------|------|------|
| 网格大小 | float | 二维网格单元边长（米） |

**适用场景**：

- 地面点粗提取
- 数字高程模型（DEM）的快速生成

### 局部最大值移除 (LocalMaximum)

移除局部邻域内的最高点，常用于去除植被顶部以获取地面。

| 参数 | 类型 | 说明 |
|------|------|------|
| 搜索半径 | float | 局部邻域搜索半径（米） |

**适用场景**：

- 植被点移除
- 地面滤波预处理

## 使用方法

### 基本操作流程

1. 在左侧树视图中选中目标点云
2. 菜单栏：`滤波` 打开滤波器工具面板
3. 从下拉列表中选择滤波器类型
4. 配置滤波器参数
5. 点击 **预览** 查看滤波效果（不修改原始数据）
6. 调整参数直到满意
7. 点击 **应用** 执行滤波，生成新的点云结果

### 参数调整建议

:::warning[预览功能]
建议在应用前使用预览功能确认效果，避免参数设置不当导致有效数据丢失。预览模式下可以反复调整参数而不影响原始数据。

:::
:::tip[组合使用]
多种滤波器可以组合使用以达到最佳效果。典型工作流：
1. 使用 **直通滤波** 粗略裁剪到感兴趣区域
2. 使用 **统计离群点移除** 清除噪声
3. 使用 **体素降采样** 统一密度

:::
## 性能说明

所有滤波算法在后台线程中执行，不会阻塞主界面。对于大规模点云：

| 点云规模 | 建议策略 |
|----------|---------|
| < 100 万点 | 所有滤波器均可直接使用 |
| 100-1000 万点 | 建议先降采样再精细滤波 |
| > 1000 万点 | 先用直通滤波裁剪区域，再进行其他滤波 |
