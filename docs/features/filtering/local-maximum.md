# 局部最大值 (Local Maximum)

局部最大值滤波是一种基于局部邻域搜索的特征提取方法。算法对点云中的每个点，在指定的二维或三维搜索半径范围内查找邻居，如果该点的 Z 值（或指定字段值）在其所有邻居中最大，则将其保留。该方法常用于提取局部高程最大的点，适合检测树冠顶点、电线高点、建筑物屋顶峰值等。

## 入口路径

`滤波 > 局部最大值 (Local Maximum)`

## 参数说明

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| 搜索半径 (Radius) | float | 0.5 | 用于查找局部邻居的搜索半径（单位：米）。在该范围内，只有 Z 值最大的点被保留 |
| 比较字段 | string | z | 用于比较的字段，默认为高程值 `z`。可设置为其他标量字段 |
| 使用二维投影 | bool | true | 是否仅在 X-Y 平面上搜索邻居（忽略 Z 方向的距离）。通常用于地面以上特征的提取 |

## 操作步骤

1. 在点云列表中选中目标点云对象
2. 点击菜单 `滤波 > 局部最大值 (Local Maximum)`
3. 设置**搜索半径**
4. 根据需要选择是否使用**二维投影**模式
5. 点击**确定**执行滤波
6. 滤波结果作为新点云对象添加到列表中

## 参数调优建议

- **搜索半径**应大于待提取目标的典型尺寸：
  - 树冠顶点检测：建议使用树冠平均半径的 1 ~ 2 倍（如 `1.0 ~ 3.0` 米）
  - 电线高点检测：`0.2 ~ 0.5` 米
  - 建筑物屋顶特征提取：`1.0 ~ 5.0` 米
- 搜索半径过小会提取出过多的局部极值点（噪声敏感），过大会遗漏真实的局部高点

## 适用场景

- **树冠顶点检测**：从机载激光雷达点云中识别单棵树的顶部位置
- **植被冠层高度模型 (CHM) 生成**：提取冠层顶部点用于生成 CHM
- **建筑物屋顶特征提取**：识别屋顶的脊线和峰值点
- **电力线路检测**：提取电线的最高点用于安全距离分析

!!! note "提示"
    局部最大值滤波对噪声较为敏感。建议在执行此操作前先进行统计离群点移除或半径离群点移除，去除高空飘散的噪声点，避免噪声点被误识别为局部最大值。

!!! tip "技巧"
    对于树木检测应用，可以先使用网格最小值提取地面，计算每个点到地面的归一化高度（nDSM），然后在 nDSM 基础上执行局部最大值滤波，效果通常优于直接在原始高程上操作。
