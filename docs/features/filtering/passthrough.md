# 直通滤波 (PassThrough)

直通滤波是一种基于指定维度范围的点云裁剪方法。用户选择一个坐标轴字段（如 X、Y、Z），并设定该轴向上的最小值和最大值，落在范围之外的点将被移除。该方法基于 PCL 的 `PassThrough` 滤波器实现，适用于快速裁剪点云中不需要的区域。

## 入口路径

`滤波 > 直通滤波 (PassThrough)`

## 参数说明

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| 字段名称 (Field) | string | z | 用于过滤的坐标轴字段，可选 `x`、`y`、`z` |
| 范围下限 (Min) | float | 0.0 | 保留范围的起始值（包含） |
| 范围上限 (Max) | float | 1.0 | 保留范围的结束值（包含） |
| 反转 (Negative) | bool | false | 勾选后，保留范围**之外**的点将被保留，范围内的点被移除 |
| 保持有序 (Keep Organized) | bool | false | 勾选后，被移除的点用 NaN 填充，保持点云结构不变 |

## 操作步骤

1. 在点云列表中选中目标点云对象
2. 点击菜单 `滤波 > 直通滤波 (PassThrough)`
3. 在参数对话框中选择**字段名称**（X/Y/Z 轴）
4. 设置**范围下限**和**范围上限**
5. 如需保留范围外的点，勾选**反转**
6. 点击**确定**执行滤波
7. 滤波结果作为新点云对象添加到列表中

## 适用场景

- **高程裁剪**：选择 Z 字段，移除高于或低于特定高程的点（如去除地面以下的噪声点或建筑物以上的飘点）
- **区域截取**：选择 X 或 Y 字段，截取感兴趣区域（ROI）内的点
- **预处理去噪**：移除扫描范围之外的明显离群点

:::tip[技巧]
可以通过多次使用直通滤波并选择不同的字段（如先按 X 裁剪再按 Y 裁剪）来实现矩形区域的精确截取。也可以配合**反转**选项，先保留目标区域再做后续处理。

:::
:::note[提示]
设置范围前，建议先在三维视窗中观察点云的空间分布，了解各轴的大致取值范围，避免因范围设置不当导致误删大量有效点。可通过点云属性面板查看包围盒（Bounding Box）信息来辅助确定合理的范围值。


:::
