# 统计离群点移除 (Statistical Outlier Removal)

统计离群点移除是一种基于局部点密度统计的噪声去除方法。对于点云中的每个点，算法计算它到其 K 个最近邻点的平均距离，然后根据全局平均距离和标准差来判断该点是否为离群点。如果某点到邻居的平均距离超过阈值（均值加上标准差的倍数），则该点被判定为离群点并移除。该方法基于 PCL 的 `StatisticalOutlierRemoval` 滤波器实现。

## 入口路径

`滤波 > 统计离群点移除 (Statistical Outlier Removal)`

## 参数说明

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| 邻居数 (MeanK) | int | 50 | 用于计算每个点平均距离的最近邻点数。值越大，统计估计越平滑，对局部密度变化的容忍度越高 |
| 标准差倍数 (StddevMulThresh) | float | 1.0 | 判定阈值 = 全局均值距离 + 该值 x 全局标准差。值越小，过滤越严格，被移除的点越多 |
| 反转 (Negative) | bool | false | 勾选后保留离群点，移除内点 |

## 操作步骤

1. 在点云列表中选中目标点云对象
2. 点击菜单 `滤波 > 统计离群点移除 (Statistical Outlier Removal)`
3. 在参数对话框中设置**邻居数**和**标准差倍数**
4. 点击**确定**执行滤波
5. 滤波结果作为新点云对象添加到列表中

## 参数调优建议

- **标准差倍数**是最关键的参数：
  - `1.0`：严格过滤，可能误删部分有效点
  - `2.0`：推荐初始值，适合大多数场景
  - `3.0` 及以上：宽松过滤，仅去除极端离群点
- **邻居数**影响统计稳定性：
  - 密集点云建议 `30 ~ 100`
  - 稀疏点云建议 `10 ~ 30`
  - 值过大会使统计结果趋于全局平均，降低局部敏感性

## 适用场景

- **扫描噪声去除**：移除激光雷达或结构光扫描中产生的飘散噪声点
- **多站拼接后处理**：消除不同站点拼接误差导致的散乱点
- **地面点云清洗**：去除地面点云中的车辆、行人等非地面干扰点
- **配准前预处理**：去除离群点可提高配准算法的精度和稳定性

!!! warning "注意"
    统计离群点移除需要对每个点进行 K 近邻搜索，计算复杂度较高。对于超大规模点云（千万级以上），建议先进行体素降采样再执行此操作，或使用空间索引（Octree）加速的半径离群点移除作为替代方案。

!!! tip "技巧"
    如果不确定合适的标准差倍数，可以先使用较大的值（如 5.0）执行一次，观察被移除的点是否为明显的噪声，然后逐步缩小倍数反复试验，直至找到合适的阈值。
