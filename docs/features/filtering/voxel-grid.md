# 体素降采样 (Voxel Grid)

体素降采样是一种经典的三维点云降采样方法。其原理是将三维空间划分为等大的立方体网格（体素），在每个体素内部用所有点的重心来代替该体素中的全部点，从而在保留点云整体几何形态的同时大幅减少点的数量。

该方法基于 PCL 的 `VoxelGrid` 滤波器实现，适用于需要对大规模点云进行快速降采样的场景。

## 入口路径

`滤波 > 体素降采样 (Voxel Grid)`

## 参数说明

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| 体素大小 (Leaf Size) | float | 0.01 | 体素立方体的边长（单位：米）。值越大，降采样越激进，点云越稀疏 |
| 体素大小 X | float | 0.01 | X 方向体素边长。勾选"独立轴设置"后可单独配置 |
| 体素大小 Y | float | 0.01 | Y 方向体素边长。勾选"独立轴设置"后可单独配置 |
| 体素大小 Z | float | 0.01 | Z 方向体素边长。勾选"独立轴设置"后可单独配置 |
| 独立轴设置 | bool | false | 是否对 X/Y/Z 三个轴分别设置不同的体素大小 |
| 下采样方式 | enum | 质心 | 质心：用体素内所有点的重心替代；随机：从体素内随机选取一个点 |

## 操作步骤

1. 在点云列表中选中目标点云对象
2. 点击菜单 `滤波 > 体素降采样 (Voxel Grid)`
3. 在弹出的参数对话框中设置**体素大小**
4. 如需对三个轴设置不同的精度，勾选**独立轴设置**并分别填入 X/Y/Z 方向的体素边长
5. 点击**确定**执行降采样
6. 降采样结果将作为新的点云对象添加到点云列表中，原始点云保持不变

## 适用场景

- **大规模点云预处理**：对动辄数百万乃至数千万点的原始点云进行快速精简，为后续处理降低计算开销
- **配准前的点云归一化**：使源点云与目标点云的密度趋于一致，提高配准精度
- **可视化优化**：降低渲染负载，提升交互流畅度
- **曲面重建前处理**：消除点云密度不均匀的影响，使重建结果更加平滑

!!! note "提示"
    体素大小直接影响输出点云的点数和保留的几何细节程度。一般建议先使用较大的值（如 0.05）进行粗略降采样以估算效果，再逐步缩小至所需精度。对于地面点云，通常 0.02 ~ 0.05 即可满足大多数应用；对于精细扫描件，可能需要 0.001 ~ 0.005。

!!! warning "注意"
    体素大小设置过小会导致降采样效果不明显，点数几乎不变；设置过大则可能丢失重要的几何特征（如尖锐边缘、细小结构）。请根据点云的实际空间范围和精度要求合理选择。
