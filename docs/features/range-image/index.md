# 深度图

深度图 (Range Image) 是将 3D 点云按照传感器视角投影到 2D 图像平面上，每个像素存储该方向上最近点的距离值。PointWorks 提供深度图生成、可视化和导出功能。

## 入口路径

**菜单**: Tools > Range Image

**源码**: `src/tool/rangeimage.h`

**算法**: `libs/algorithm/keypoints.h`（NARF 关键点依赖深度图）

## 参数说明

| 参数 | 控件 | 说明 |
|------|------|------|
| Angular Resolution | DoubleSpinBox | 角度分辨率（度/像素），控制深度图的精细程度 |
| Horizontal FOV | DoubleSpinBox | 水平视场角（度） |
| Vertical FOV | DoubleSpinBox | 垂直视场角（度） |
| Show Point Cloud | CheckBox | 是否在深度图上叠加显示原始点云 |

### 交互操作

| 操作 | 说明 |
|------|------|
| 鼠标滚轮 | 缩放深度图 |
| 鼠标拖拽 | 平移深度图 |
| Export | 导出深度图为图片文件 |
| Close | 关闭深度图窗口 |

## 操作步骤

1. 选中点云后打开 Range Image 工具
2. 设置 **Angular Resolution**（角度分辨率）：
   - 较小的值（如 0.5 度）生成更精细的深度图，但图像更大
   - 较大的值（如 2 度）生成较粗糙的深度图，计算更快
3. 设置水平/垂直视场角范围
4. 深度图自动根据当前相机视角 (Viewer Pose) 生成
5. 旋转 3D 视图时深度图会自动更新（带防抖延迟）
6. 使用鼠标滚轮缩放和拖拽平移查看细节
7. 点击 **Export** 导出为图片

> 深度图的视角 (Viewer Pose) 与 3D 视图的相机位置同步。旋转 3D 视图后，深度图会自动根据新的视角重新生成。防抖定时器 (Debounce Timer) 确保连续旋转时不会频繁重建。

## 输出

深度图以灰度图像的形式显示在对话框中，距离越近颜色越亮，距离越远颜色越暗。导出时保存为 PNG 或 BMP 格式。

## 应用场景

| 场景 | 说明 |
|------|------|
| 可见性分析 | 从特定视角观察点云的可见区域 |
| NARF 关键点 | 深度图是 NARF 关键点检测的前置条件 |
| 快速浏览 | 在 2D 图像中快速了解点云的空间分布 |
| 质量检查 | 检查扫描覆盖率和遮挡区域 |
