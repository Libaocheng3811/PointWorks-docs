# 包围盒信息

查看和显示点云或网格的包围盒 (Bounding Box)。包围盒是包含所有点的最小轴对齐长方体，提供点云的空间范围信息。

## 入口路径

**菜单**: Edit > Bounding Box

**源码**: `src/edit/boundingbox.h`

## 功能说明

### 包围盒类型

| 类型 | 说明 |
|------|------|
| 轴对齐包围盒 (AABB) | 与坐标轴对齐的长方体，计算快速 |
| 有向包围盒 (OBB) | 根据点云形状旋转的长方体，更紧凑 |

### 显示信息

包围盒提供以下空间信息：

| 属性 | 说明 |
|------|------|
| Min (X, Y, Z) | 包围盒最小角点坐标 |
| Max (X, Y, Z) | 包围盒最大角点坐标 |
| Size (X, Y, Z) | 各轴方向的尺寸 |
| Center | 包围盒中心坐标 |
| Volume | 包围盒体积 |

## 操作步骤

1. 选中点云或网格后打开 Bounding Box 工具
2. 选择包围盒类型
3. 点击 **Preview** 在 3D 视图中预览包围盒线框
4. 点击 **Apply** 确认显示包围盒
5. 点击 **Reset** 移除包围盒显示

> 通过 `m_box_map` 记录每个点云的包围盒状态，切换选中的点云时会自动更新包围盒显示。

> 包围盒的显示颜色可通过 Options > Display Settings > Colors 页面进行自定义。
