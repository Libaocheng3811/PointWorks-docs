# 颜色设置

设置点云、法线和网格模型的显示颜色。支持统一颜色设置、标量场着色和坐标轴分量着色等多种模式。

## 入口路径

**菜单**: Edit > Colors

**源码**: `src/edit/color.h`

## 功能说明

### 点云颜色

| 模式 | 说明 |
|------|------|
| RGB 颜色 | 选择一个固定颜色应用到所有点 |
| Scalar Field | 按标量场值进行色度映射着色 |
| 坐标轴分量 | 按 X/Y/Z 坐标值进行颜色渐变 |

### 法线颜色

当点云包含法线数据时，可以单独设置法线的显示颜色。

### 网格颜色

对加载的网格模型设置统一颜色。

## 参数说明

| 参数 | 控件 | 说明 |
|------|------|------|
| Color Picker | 颜色选择器 | 选择 RGB 颜色 |
| Field Selector | 下拉框 | 选择标量场进行着色 |
| Axis Buttons | 按钮组 | 按 X/Y/Z 轴坐标值着色 |
| Target Selector | 下拉框 | 选择着色目标（点云 / 法线 / 网格） |

## 操作步骤

1. 选中点云或网格后打开 Colors 工具
2. 选择着色目标（根据选中节点类型自动切换可用选项）
3. 选择着色方式：
   - **固定颜色**: 在颜色选择器中选取颜色，点击 Apply
   - **标量场着色**: 从下拉框选择已计算的标量字段
   - **坐标轴着色**: 点击 X/Y/Z 按钮按坐标值着色
4. 点击 **Apply** 应用颜色设置
5. 点击 **Reset** 恢复默认颜色

> 标量场着色需要先通过距离计算等操作生成标量数据。若下拉框为空，说明当前点云没有标量场。

> 颜色设置是显示级别的修改，不会修改原始数据的颜色值。如需永久修改颜色，请使用 Apply 并保存。
