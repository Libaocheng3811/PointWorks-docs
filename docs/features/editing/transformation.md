# 坐标变换

对点云进行平移、旋转和仿射变换。支持通过数值参数或 4x4 变换矩阵进行精确变换，并可保存/加载变换矩阵。

## 入口路径

**菜单**: Edit > Transformation

**源码**: `src/edit/transformation.h`

## 变换方式

### 欧拉角参数

通过平移和旋转参数定义变换：

| 参数 | 说明 |
|------|------|
| Translate X | X 方向平移量 |
| Translate Y | Y 方向平移量 |
| Translate Z | Z 方向平移量 |
| Rotate X (Roll) | 绕 X 轴旋转角度（度） |
| Rotate Y (Pitch) | 绕 Y 轴旋转角度（度） |
| Rotate Z (Yaw) | 绕 Z 轴旋转角度（度） |

### 变换矩阵

直接输入 4x4 仿射变换矩阵。矩阵格式为 4 行，每行 4 个数值，空格或逗号分隔。

```
| r11  r12  r13  tx |
| r21  r22  r23  ty |
| r31  r32  r33  tz |
|  0    0    0   1  |
```

## 操作步骤

1. 选中点云后打开 Transformation 工具
2. 输入变换参数（欧拉角或矩阵）
3. 点击 **Preview** 实时预览变换效果（不修改数据）
4. 确认效果后：
   - **Apply**: 直接修改当前点云的坐标
   - **Add**: 创建变换后的新点云副本（前缀 `transed-`），保留原始点云
5. 点击 **Reset** 恢复预览前的状态

> 变换历史会记录在 `m_trans_map` 中，可以在同一点云上叠加多次变换。每次 Add 操作会生成新的点云节点。

> 使用 **Apply** 会直接修改原始点云坐标，此操作不可撤销。建议先用 **Preview** 确认效果，或使用 **Add** 保留原始数据。
