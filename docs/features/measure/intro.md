# 测量工具

在 3D 视图中通过点拾取测量点云上两点之间的距离。支持多组测量、标注显示和结果导出。

## 入口路径

**菜单**: Tools > Measure

**源码**: `src/tool/measure.h`

## 功能说明

### 测量模式

| 操作 | 说明 |
|------|------|
| 单击拾取 | 点击第一个点，再点击第二个点完成一组测量 |
| 拖拽拾取 | 按住鼠标拖拽选取起止点 |

### 测量信息

每组测量记录以下数据：

| 属性 | 说明 |
|------|------|
| Distance | 两点之间的欧氏距离 |
| dX, dY, dZ | 三个坐标轴方向的分量 |
| Start Point | 起点坐标 |
| End Point | 终点坐标 |

### 显示选项

| 选项 | 说明 |
|------|------|
| Show Labels | 显示/隐藏距离标注文字 |
| Show Arrows | 显示/隐藏连接箭头 |
| Precision | 小数位数（0~6） |

### 管理操作

| 操作 | 说明 |
|------|------|
| Start/Stop | 开始或停止测量模式 |
| Clear All | 清除所有测量记录 |
| Delete Selected | 删除表格中选中的测量记录 |
| Export | 将测量结果导出为文本文件 |

## 操作步骤

1. 选中点云后打开 Measure 工具
2. 点击 **Start** 进入测量模式
3. 在 3D 视图中点击第一个测量点（光标处会显示标记）
4. 移动鼠标到第二个位置并点击
5. 距离值自动计算并显示在标注和表格中
6. 重复步骤 3~5 进行多组测量
7. 可在表格中选择某行后点击 **Delete Selected** 删除
8. 点击 **Export** 导出所有测量数据
9. 点击 **Stop** 退出测量模式

> 测量模式激活时，鼠标左键用于拾取点。如需旋转视图，请使用鼠标中键或右键。

> 测量精度取决于点云的密度。在稀疏区域，拾取到的可能是最近点而非精确位置。可调整 Precision 控制显示的小数位数。
