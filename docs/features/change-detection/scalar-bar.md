# 色度条与标量场可视化

距离计算等操作会在点云上生成标量场数据。PointWorks 提供色度条 (Scalar Bar) 功能，将标量值映射为颜色，直观展示距离、强度等标量场的空间分布。

## 功能说明

### 标量场

标量场是存储在点云中的逐点数值数据。以下操作会生成标量场：

- 云对云距离计算 (C2C)
- 云对网格距离计算 (C2M)
- 云对几何基元距离计算 (C2P)

标量场通过 `Cloud::m_scalar_cache` 管理，支持自定义字段名称。

### 色度条 (Scalar Bar)

色度条使用 Jet 色带将标量值映射为颜色：

- 蓝色 (Blue) 对应最小值
- 绿色 (Green) 对应中间值
- 红色 (Red) 对应最大值

色度条在 3D 视图底部显示，标注数值范围。

## 使用方式

### 自动着色

在距离计算对话框中勾选 **Color Map**，计算完成后自动对点云进行色度着色。

### 手动控制

通过 Edit > Colors 工具可以：

- 选择已计算的标量场进行着色
- 设置色度映射范围
- 切换不同的颜色方案

## 操作步骤

1. 执行距离计算（如 C2C 或 C2M）
2. 勾选 **Color Map** 选项
3. 计算完成后，点云自动按距离值着色
4. 色度条显示在 3D 视图中，标注最小值和最大值
5. 在 **Field Name** 中可自定义标量场名称，便于后续查找

> 色度条的颜色范围默认根据标量值的实际分布自动确定。超出最大距离限制的点以灰色或透明显示（标记为 NaN）。

## 应用场景

| 场景 | 说明 |
|------|------|
| 变化检测 | 通过色度条直观比较两个时期的变化量 |
| 质量检测 | 显示点云与设计模型之间的偏差分布 |
| 强度可视化 | 按激光反射强度值着色 |
| 高程着色 | 按 Z 坐标值进行高程色带映射 |
