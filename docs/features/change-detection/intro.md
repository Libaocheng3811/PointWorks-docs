# 变化检测

变化检测是 PointWorks 中用于比较两个时期点云数据差异的功能模块。通过计算参考点云与比较点云之间的逐点距离，生成距离标量场并支持色度条可视化，直观展示发生变化（新增、消失、位移）的区域。

## 功能概览

| 工具 | 菜单路径 | 说明 |
|------|----------|------|
| 云对云距离 | Distance > Cloud to Cloud | 两个点云之间的距离计算 |
| 云对网格距离 | Distance > Cloud to Mesh | 点云到三角网格的距离计算 |
| 变化检测插件 | Plugins > Change Detection | 专为变化检测场景设计的集成工具 |

## 工作流程

1. 加载两个时期的点云（参考云和比较云）
2. 确保两个点云已配准到同一坐标系
3. 选择距离计算方法并设置参数
4. 执行距离计算
5. 通过色度条 (Scalar Bar) 可视化距离分布
6. 根据距离阈值判断变化区域

> 变化检测的前提是两个点云处于同一坐标系下。若坐标不一致，请先使用 Registration 工具进行配准。
