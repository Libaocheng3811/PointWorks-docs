# 导入与导出

本文介绍如何在 PointWorks 中导入和导出点云与网格文件。

## 导入文件

### 基本操作

1. 通过以下任一方式打开文件：
    - 菜单栏：`文件 > 打开` 或 `文件 > 导入`
    - 快捷键：`Ctrl + O`
    - 拖放：将文件直接拖入主窗口
2. 在文件选择对话框中选择一个或多个文件
3. 根据文件类型，可能弹出额外的配置对话框（见下方说明）
4. 文件开始加载，进度条显示加载状态
5. 加载完成后，点云将出现在左侧树视图中，三维视图中自动显示

### TXT 文件导入配置

导入 TXT/XYZ/ASC 文件时，会弹出导入配置对话框：

| 设置项 | 说明 |
|--------|------|
| 分隔符 | 列分隔字符（空格、逗号、制表符等） |
| 跳过行数 | 文件开头需要跳过的行数（如表头） |
| 字段映射 | 将文件列映射到点属性（X、Y、Z、R、G、B、Intensity 等） |

!!! tip "字段映射"
    在字段映射对话框中，你可以预览文件的前几行数据，直观地将每一列分配到对应的属性字段。未映射的列将被忽略。

### 全局坐标偏移

当加载的点云使用大坐标系（如 UTM 坐标 x=500000, y=4000000）时，PointWorks 会自动检测并弹出全局偏移设置对话框：

| 选项 | 说明 |
|------|------|
| 自动计算偏移 | 根据点云质心自动计算偏移值 |
| 手动设置偏移 | 用户输入 X、Y、Z 偏移量 |
| 记住此偏移 | 将偏移值保存，后续加载同文件时自动应用 |
| 不使用偏移 | 保持原始坐标 |

!!! warning "大坐标问题"
    如果跳过全局偏移设置，大坐标值在 GPU 渲染时可能导致浮点精度问题，表现为点云显示抖动或异常。建议始终启用自动偏移。

### 批量导入

支持同时选择多个文件进行批量导入。批量导入时：

- 各文件按顺序依次加载
- 进度条显示总体进度
- 所有文件加载完成后统一刷新视图
- 每个文件在树视图中作为独立节点显示

## 导出文件

### 基本操作

1. 在左侧树视图中选中要导出的点云或网格
2. 通过以下任一方式触发导出：
    - 菜单栏：`文件 > 导出`
    - 右键菜单：在树视图节点上右键，选择 `导出`
    - 快捷键：`Ctrl + S`（保存当前选中项）
3. 选择目标格式和文件路径
4. 配置格式特定的导出选项
5. 点击保存

### TXT 文件导出配置

导出为 TXT 格式时，可配置以下选项：

| 设置项 | 说明 |
|--------|------|
| 分隔符 | 输出列之间的分隔字符 |
| 字段选择 | 选择要导出的属性列（X、Y、Z、R、G、B、Intensity 等） |
| 精度 | 坐标值的小数位数 |
| 表头 | 是否输出列名表头 |

### 保存操作

| 操作 | 快捷键 | 说明 |
|------|--------|------|
| 保存 | `Ctrl + S` | 保存选中点云到原文件 |
| 另存为 | `Ctrl + Shift + S` | 选择新路径和格式保存 |
| 全部保存 | -- | 保存所有已修改的点云 |

!!! tip "格式转换"
    你可以通过 "另存为" 功能在不同格式之间转换。例如，加载 LAS 文件后导出为 PCD 格式，即可实现 LAS 到 PCD 的转换。

## 支持的导入导出格式矩阵

| 格式 | 导入 | 导出 | 备注 |
|------|:----:|:----:|------|
| LAS | Yes | Yes | 完整属性支持 |
| LAZ | Yes | Yes | LAS 压缩格式 |
| PLY | Yes | Yes | 支持 ASCII 和二进制模式 |
| PCD | Yes | Yes | PCL 原生格式 |
| E57 | Yes | -- | 仅导入 |
| TXT/XYZ/ASC | Yes | Yes | 自定义字段映射 |
| OBJ | Yes | -- | 含纹理检测 |
| STL | Yes | -- | ASCII/二进制 |
| VTK | Yes | -- | VTK PolyData |
| IFS | Yes | -- | 内部格式 |
