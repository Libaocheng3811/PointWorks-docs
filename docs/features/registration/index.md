# 点云配准 (Registration)

点云配准是将两个或多个处于不同坐标系下的点云对齐到统一坐标系的过程。在实际应用中，由于扫描仪视角限制或多次扫描的需要，同一场景的不同站点或不同视角的点云之间存在空间变换关系（平移和旋转）。配准的目标就是求解这种变换关系，使不同点云之间达到最佳重合。

PointWorks 提供了完整的点云配准工作流，覆盖从粗配准到精配准的各个环节。

## 入口路径

`配准` 菜单栏

## 配准工作流

典型的点云配准流程包括以下步骤：

```
粗配准（获取初始变换） --> 精配准（优化变换） --> 结果验证
```

### 第一步：粗配准

粗配准的目的是为两片点云提供一个较好的初始对齐位置。精配准算法（如 ICP）需要良好的初始位置才能收敛到正确结果，因此粗配准是整个流程的关键前提。

PointWorks 提供以下粗配准方式：

| 方法 | 适用场景 | 是否需要人工交互 |
|------|---------|-----------------|
| [中心对齐](align-by-centers.md) | 两片点云位置差异不大，仅需简单平移 | 否 |
| [全局配准 (IA-RANSAC)](global-registration.md) | 两片点云存在任意旋转和平移 | 否 |
| [点对配准](point-pairs.md) | 已知对应点关系，需要精确初始对齐 | 是（手动选点） |

### 第二步：精配准

在粗配准获得初始变换后，使用精配准算法对变换参数进行迭代优化，使两片点云之间的对齐误差最小化。

| 方法 | 适用场景 |
|------|---------|
| [精配准 (ICP)](fine-registration.md) | 在良好初始对齐基础上进行高精度优化 |

### 第三步：结果验证

配准完成后，建议进行以下验证：

1. **视觉检查**：在三维视窗中将配准后的两片点云叠加显示，使用不同颜色区分，观察重叠区域的吻合程度
2. **误差统计**：查看配准结果中的均方误差（MSE）或点到面距离分布
3. **特征对齐**：检查两片点云中的相同特征（如建筑物角点、路面标线）是否精确对齐

## 基本概念

| 术语 | 说明 |
|------|------|
| 源点云 (Source) | 需要被变换对齐到目标点云的点云 |
| 目标点云 (Target) | 作为参考基准的点云，配准过程中保持不动 |
| 变换矩阵 (Transformation) | 描述源点云到目标点云的刚体变换（4x4 矩阵），包含旋转和平移 |
| 收敛 (Convergence) | 迭代优化过程中误差不再显著减小的状态 |
| 重叠率 (Overlap) | 两片点云之间实际重叠区域占总面积的比例 |

## 操作前提

- 至少加载两片点云到点云列表中
- 确认哪片点云作为**目标点云**（参考基准）
- 如果使用 ICP 精配准，两片点云需要有较好的初始对齐

!!! note "提示"
    配准前建议先对两片点云分别进行降采样和去噪处理。降采样可以大幅减少配准的计算时间，去噪则能提高配准的精度和稳定性。推荐使用体素降采样 + 统计离群点移除的组合预处理流程。

!!! warning "注意"
    ICP 精配准只能在初始对齐较好的情况下收敛。如果两片点云之间的旋转角度超过约 30 度或平移距离过大，ICP 可能收敛到错误的局部最优解。在这种情况下，请先使用全局配准 (IA-RANSAC) 或点对配准获取初始变换。
