# 点对配准 (Point Pairs Registration)

点对配准是一种基于人工选取对应点对的交互式配准方法。用户分别在源点云和目标点云上选取若干对对应点（至少 3 对），系统根据这些对应点对计算出最优的刚体变换矩阵，将源点云对齐到目标点云。该方法适用于两片点云之间没有明显几何特征无法自动配准的场景，或需要精确控制配准对应关系的场合。

## 入口路径

`配准 > 点对配准 (Point Pairs)`

## 参数说明

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| 最少点对数 | int | 3 | 计算刚体变换所需的最少对应点对数。3 对即可确定一个刚体变换，更多点对可以提高精度 |
| 选点模式 | enum | 逐点选取 | 交互选点方式：`逐点选取`（交替在两片点云上选点）或 `分批选取`（先在一片上选完再到另一片选） |

## 操作步骤

### 第一步：进入点对配准模式

1. 在点云列表中选中**两片**点云（按住 Ctrl 键多选）
2. 点击菜单 `配准 > 点对配准 (Point Pairs)`
3. 进入点对选点交互模式，三维视窗中出现选点工具栏

### 第二步：选取对应点对

4. 在三维视窗中，使用鼠标点击选取源点云上的一个点（点会被高亮标记并编号）
5. 切换到目标点云，选取与源点云上该点对应的同一空间位置点
6. 重复步骤 4-5，直到选取至少 **3 对**对应点
7. 建议选取的对应点尽量满足：
   - 分布在点云的不同区域（避免所有点对集中在局部）
   - 选择特征明显的点（如角点、边缘交叉点）
   - 点对之间的距离适中，避免过近导致变换估计不稳定

### 第三步：执行配准

8. 确认所有对应点对选取完毕后，点击工具栏上的**执行配准**按钮
9. 系统计算最优刚体变换并应用到源点云
10. 配准结果（变换矩阵、残差信息）在日志面板中输出
11. 变换后的源点云作为新对象添加到列表中

### 对应点对的选择建议

```
    源点云                     目标点云
   +-----------+             +-----------+
   |  1        |             |  1'       |
   |     2     |    ====>    |     2'    |
   |        3  |             |        3' |
   |  4        |             |  4'       |
   +-----------+             +-----------+
   
   1 <-> 1'  (左上角)
   2 <-> 2'  (中心区域)
   3 <-> 3'  (右下角)
   4 <-> 4'  (左下角)
```

- 对应点应尽量在空间上均匀分布
- 避免三点共线，否则变换矩阵求解不稳定
- 选择不易混淆的明显特征点

## 适用场景

- **缺乏几何特征的场景**：对于平坦表面、规则几何体等缺乏特征描述子的点云，自动配准方法可能失败，此时点对配准是可靠的替代方案
- **跨传感器配准**：来自不同类型传感器（如激光雷达与摄影测量）的点云之间配准
- **已知对应关系的精确配准**：当操作者对两片点云的对应关系有明确认知时
- **自动配准失败的补救**：当 IA-RANSAC 全局配准结果不理想时，使用点对配准作为粗配准

## 局限性

- 需要人工交互，无法批量自动化处理
- 配准精度依赖于操作者选取对应点的准确性
- 对于大型点云，在三维视窗中精确定位点可能比较困难

:::tip[技巧]
选点时建议放大视图并旋转到合适的观察角度。可以使用鼠标滚轮缩放至点级别精度，确保选中的是准确的对应点。选点后可以通过查看点对残差来判断选取质量，残差较大的点对可能是误选，可以删除后重新选取。

:::
:::note[提示]
点对配准通常作为粗配准使用。完成点对配准后，建议再执行一次[精配准 (ICP)](fine-registration)以进一步优化对齐精度。点对配准提供初始变换，ICP 在此基础上进行迭代优化，两者配合可以达到最佳配准效果。

:::
:::warning[注意]
选取的对应点对必须是两片点云中**同一物理位置**的点。如果选错对应关系（例如将源点云的 A 位置匹配到目标点云的 B 位置），计算出的变换矩阵将是错误的。请在选点时仔细确认对应关系的正确性。


:::
