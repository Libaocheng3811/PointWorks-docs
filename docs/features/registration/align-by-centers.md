# 中心对齐 (Align by Centers)

中心对齐是最简单的粗配准方法。它计算源点云和目标点云的几何中心（质心），然后将源点云平移，使其质心与目标点云的质心重合。该方法仅执行平移变换，不涉及旋转，因此适用于两片点云之间仅有平移偏差的场景，或作为其他配准方法的预处理步骤。

## 入口路径

`配准 > 中心对齐 (Align by Centers)`

## 参数说明

此功能无需额外参数设置。执行时自动计算两片点云的质心并完成平移对齐。

## 操作步骤

1. 在点云列表中选中**两片**点云（按住 Ctrl 键多选）
2. 点击菜单 `配准 > 中心对齐 (Align by Centers)`
3. 系统自动执行以下操作：
   - 计算源点云的质心坐标
   - 计算目标点云的质心坐标
   - 将源点云平移至目标点云质心位置
4. 对齐后的源点云作为新点云对象添加到列表中

## 适用场景

- **同站重复扫描**：同一扫描位置不同时间的两次扫描，仅有微小平移偏差
- **配准预处理**：在执行 ICP 精配准前，先将两片点云的大致位置对齐
- **快速预览**：快速查看两片点云在质心重合时的相对位置关系

## 局限性

- 仅进行平移，不处理旋转差异
- 对于姿态差异较大的两片点云，中心对齐后仍可能有较大偏差
- 点云内部密度不均匀时，质心位置可能偏离几何中心

:::note[提示]
中心对齐通常作为 ICP 精配准的前置步骤使用。先进行中心对齐消除平移偏差，再执行 ICP 优化旋转和残余平移，可以获得更稳定的配准结果。

:::
:::tip[技巧]
如果两片点云不仅存在平移差异，还包含旋转差异，建议使用[全局配准 (IA-RANSAC)](global-registration)代替中心对齐，或使用[点对配准](point-pairs)进行手动精确对齐。


:::
